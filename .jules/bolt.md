\n## 2026-05-13 - Supabase Select Optimization\n**Learning:** The previous implementation used `select('*')` excessively in Next.js Server Components, which fetches unused data and bloated server memory/network payload sizes.\n**Action:** Always explicitly specify required column names (e.g., `select('id, name')`) and utilize the `{ count: 'exact', head: true }` option when only the total number of rows is needed.
